# Civil Engineer AI marketing media

Optimized illustration assets for the homepage and demo entry pages. These are
presentation-only marketing visuals. They are referenced by stable string paths
through `lib/marketingMedia.ts` and rendered with the `MarketingMedia`
component, which shows a polished fallback panel when a file is missing. Adding
or removing a file here never breaks the build.

Do not include secrets, tokens, storage keys, signed URLs, or real project data
in these images. The artwork must be illustrative review-support media only and
must not imply plan approval, certification, or any final engineering decision.

## Expected files

- `site-plan-review-hero.webp`
- `review-workflow-visual.webp`
- `technical-foundation-visual.webp`
- `human-review-boundary.webp`
- `guided-demo-journey.webp`

## Recommended dimensions and format

- Format: WebP, optimized and compressed for the web.
- Hero and section images: 16:9, around 1600px wide.
- Boundary image: 4:3 or 16:9, compressed WebP.

Keep file sizes small so the landing page stays fast.
